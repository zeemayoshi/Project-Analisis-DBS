==========================================
Required Libraries
==========================================

The following Python libraries are required to run the Bike Sharing Dataset Analysis:

- **pandas**: Data manipulation and analysis
- **streamlit**: Building interactive web applications for data analysis
- **numpy**: Numerical computations
- **matplotlib**: Data visualization
- **seaborn**: Statistical data visualization
- **tabulate**: Displaying tabular data in a readable format
- **IPython**: Interactive computing support
- **openpyxl**: Handling Excel files
- **scikit-learn**: Machine learning and data preprocessing

==========================================
Installation
==========================================

To install all required libraries, run the following command in your terminal:

pip install -r requirements.txt

Alternatively, you can install them individually:

pip install pandas streamlit numpy matplotlib seaborn tabulate IPython openpyxl scikit-learn
